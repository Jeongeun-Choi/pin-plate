# Quickstart: Native Script Map

## Prerequisites
-   Client ID in `.env.local`.

## Verification Steps

1.  **Clean Install**:
    ```bash
    pnpm install
    ```
2.  **Run Development Server**:
    ```bash
    pnpm dev:web
    ```
3.  **Visual Check**:
    -   [ ] Map loads (no React Context errors).
    -   [ ] Console has no `react-naver-maps` related errors.
    -   [ ] Map center is Seoul City Hall.
    -   [ ] Map is interactive (drag/zoom).
