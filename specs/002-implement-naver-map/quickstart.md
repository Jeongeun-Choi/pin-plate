# Quickstart: Naver Map View

## Prerequisites
-   Ensure `.env.local` has valid `NEXT_PUBLIC_NAVER_MAP_CLIENT_ID`.

## Verification Steps

1.  **Start Development Server**:
    ```bash
    pnpm dev:web
    ```
2.  **Open Browser**:
    -   Navigate to `http://localhost:3000`.
3.  **Visual Check**:
    -   [ ] Map loads without errors.
    -   [ ] Center is at Seoul City Hall.
    -   [ ] Map takes up full screen height/width.
    -   [ ] Resize window to mobile size -> Map adjusts height correctly.
